A free, fun game where you shoot baddies in space!
Or try to make them fall off!
And try not to fall off yourself!

About On the Edge

On the Edge is a fun little game where you need combine use of weapons and various gadgets to find your way to the end of each level. Is it better to just shoot or maybe push enemies or mines around? This game combines some light puzzle aspects with non-stop action.

The game was created around 2005 by a team of hobbyists (back then called Azuris Productions team) mostly to have fun and learn more about game development.

The game is programmed in C++ / DirectX 9 and requires a basic Windows PC to run.

There are a few low-res videos of the game in action
